# HimaRoLink
ブラウザのトップページに指定すると便利なリンク集型ＷＥＢサイトです。（日本人向け）

2021/12　サイト内でブラウジングができる「埋め込みアクセス機能」追加<br>
2022/03　UI大刷新、バージョン制採用、2022年3月リリースのバージョンをVer.3.0に決定<br>
2022/07　暇人が作ったサイトVer.4.0リリース（リンクをジャンル分け、リンク15件追加、メールボックス開設）<br>
